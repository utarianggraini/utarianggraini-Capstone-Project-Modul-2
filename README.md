### **SaaS Sales Data Analysis to Identify High-Potential Customer**

**1. Deskripsi Project**
Project ini bertujuan untuk menganalisis data penjualan perusahaan Software as a Service (SaaS) guna mengidentifikasi customer dengan potensi tinggi serta menemukan faktor-faktor yang memengaruhi profit dan kerugian perusahaan.

Analisis dilakukan menggunakan Python untuk data cleaning, analisis statistik, dan visualisasi, serta Tableau untuk pembuatan dashboard interaktif.

**2. Tujuan Analisis**
- Mengidentifikasi segment dan region dengan total penjualan tertinggi.
- Menentukan customer yang paling menguntungkan.
- Menganalisis hubungan antara diskon dan profit.
- Mengidentifikasi industry dengan performa terbaik dan terburuk.
- Memberikan insight bisnis untuk meningkatkan profit perusahaan.

**3. Dataset**
Nama dataset: SaaS Sales Data
Format: CSV / Excel
Sumber: Dataset simulasi penjualan SaaS
Kolom Utama: Order Date, Segment, Region, Industry, Customer, Sales, Profit, Discount.

**4. Data Cleaning (Python)**
Tahapan pembersihan data yang dilakukan:
1. Mengubah kolom tanggal ke format datetime.
2. Mengecek dan menangani missing values.
3. Menghapus data duplikat.
4. Memastikan tipe data numerik sudah sesuai.
5. Validasi nilai profit dan discount.

**5. Exploratory Data Analysis (EDA)**
Analisis eksplorasi data meliputi:
1. Statistik deskriptif (mean, median, minimum, maksimum).
2. Distribusi nilai sales dan profit.
3. Analisis total sales berdasarkan segment.
4. Analisis total profit berdasarkan region.
5. Identifikasi outlier pada profit dan discount.

**6. Data Visualization (Python)**
Visualisasi yang dibuat menggunakan Python:
1. Bar chart total sales per segment.
2. Bar chart total profit per region.
3. Scatter plot hubungan antara discount dan profit.
4. Visualisasi top customers berdasarkan profit.
5. Library yang digunakan: pandas, matplotlib, seaborn.

**7. Tableau Dashboard**
Dashboard Tableau dibuat untuk memvisualisasikan:
1. Total Sales by Segment
2. Total Profit by Region
3. Top Customers (Sales & Profit)
4. Discount vs Profit
5. Industry Performance

**8. Kesimpulan**
Project ini menunjukkan bahwa analisis data penjualan dapat membantu perusahaan SaaS dalam:
1. Mengoptimalkan strategi diskon.
2. Fokus pada customer dengan profit tinggi.
3. Mengevaluasi performa industry dan region.
4. Mendukung pengambilan keputusan berbasis data.
